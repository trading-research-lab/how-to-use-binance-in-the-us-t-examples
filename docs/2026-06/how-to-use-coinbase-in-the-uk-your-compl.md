# Coinbase UK Setup & Compliance Checklist

**Source:** How to Use Coinbase in the UK: Your Complete Setup and Trading Guide

## Account Setup Checklist
- [ ] Create Coinbase account (email + password)
- [ ] Enable two-factor authentication (2FA)
- [ ] Complete email verification
- [ ] Verify phone number
- [ ] Complete identity verification (UK passport or driver's license required)
- [ ] Provide proof of residence (utility bill, bank statement, dated within 3 months)

## UK Payment Methods & Fees

| Method | Deposit Limit | Withdrawal Speed | Typical Fees | Notes |
|--------|---|---|---|---|
| Bank Transfer (SEPA) | High | 1-3 days | 1-1.5% | Primary method for UK users |
| Debit Card | £5,000-£10,000/day | Instant | 1.5-2% | Faster but higher fees |
| Credit Card | Variable | Instant | 2-3.5% | Not recommended (cash advance fees) |
| PayPal | £1,000-£5,000 | 1-2 days | 2% | Alternative option |

## Security Best Practices (UK)
- [ ] Use 2FA (authenticator app preferred over SMS)
- [ ] Whitelist withdrawal addresses before use
- [ ] Enable login alerts
- [ ] Use strong, unique password (16+ characters, mixed case/numbers/symbols)
- [ ] Store recovery codes offline
- [ ] Never share API keys or seed phrases
- [ ] Consider using a hardware wallet for holdings >£5,000
- [ ] Review Coinbase Privacy Policy for UK GDPR compliance

## Identity Verification Requirements (UK)
1. **Primary ID:** UK Passport or Driving Licence
2. **Proof of Residence:** Any document dated within 3 months showing name and UK address
3. **Processing Time:** Usually 24-48 hours
4. **Rejection Reasons:** Blurry documents, expired ID, unreadable address

## UK Tax & Regulatory Considerations
- [ ] Keep transaction records for HMRC (Capital Gains Tax reporting)
- [ ] Log cost basis and disposal dates
- [ ] Report crypto as income if earned/staked
- [ ] Coinbase provides transaction history export for tax filing
- [ ] FCA regulates Coinbase in UK; complaint redress available via FCA

## Common Issues & Solutions
| Issue | Cause | Solution |
|---|---|---|
| Verification stuck | Poor image quality | Retake photos with good lighting, centered document |
| Card declined | Bank blocks crypto | Contact bank, whitelist Coinbase, use bank transfer instead |
| Withdrawal pending >24h | AML review | Standard for large transfers; typically resolves within 48h |
| Account restricted | Multiple failed logins | Contact support; may require additional identity verification |

## Account Setup Checklist

- [ ] Verify residency in eligible state
- [ ] Complete full KYC verification
- [ ] Enable 2FA security
- [ ] Fund account (wire transfer / crypto deposit)
- [ ] Review risk management policies
- [ ] Set position size limits
- [ ] Enable price alert notifications

## Risk Management Best Practices

1. **Position Sizing**: Start with 1-2% of account per trade
2. **Stop-Loss Orders**: Set mandatory stops on every entry
3. **Leverage Cap**: Recommended max 5:1 for beginners
4. **Daily Loss Limit**: Set a hard stop if you lose X% per session
5. **Funding Rate Monitoring**: Check funding rates before holding long positions
6. **Liquidation Price**: Always know your liquidation level before opening
